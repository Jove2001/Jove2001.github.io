    Raspberry Pi Projects

    Summary

    The Raspberry Pi is a versatile and affordable single-board computer. I've used it for a variety of projects since the 2B model for a retro gaming to a home server. It's the real 'Swiss army knife' of computing.

    My current Raspberry Pi 5 is a great platform for learning about Linux, networking, and programming. I've built several projects on it, and the best part is that I can create different configurations on SD cards. It can go from a retro-gaming console, to a desktop computer, to a home server, all with a simple swap of the SD card. It's perfect for prototyping new ideas!

    I love the FLIRC case for the Raspberry Pi 5. It's a solid aluminum case that acts as a heat sink for the CPU. It's very sturdy and a great way to keep the Pi safe and cool.

    Favourite configurations

    • RetroPie with 3000+ games and four SNES style controllers for multiplayer mayhem.
    • Raspberry Pi OS Desktop for general purpose computing and Python programming.
    • Ubuntu Server 24.04 LTS with Docker
