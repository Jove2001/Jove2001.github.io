    DriveHub

    This was my final project for my Bachelor of Information Technology. It's a web application that allows users to reserve a car and collect it from a pod. DriveHub is a complete on-demand car rental solution, and includes:

  • An admin portal.
  • Payment processing with Stripe
  • Email verification with SendGrid
  • A complete test suite

    • It was built using ASP.NET Core and SQL Server 2022. I'm very proud of this project. We received a High Distinction for it. I led the development team, developed the technical solutions, coded the back-end, designed the database, and managed the deployments. The code is available on GitHub.
